## Installation

To install this application, follow these steps:

1. Clone the repository:

   ```bash
   git clone https://github.com/parzi-val/grocery_app_apis
   ```

2. Navigate into the project directory:

   ```bash
   cd grocery_app_apis
   ```

3. Install the dependencies:

   ```bash
   npm install
   ```
